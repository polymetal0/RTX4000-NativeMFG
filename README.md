# Native MFG in RTX 4000 GPUs

This is a fork of DLSSSpoofer that's been modified to spoof a Nvidia Blackwell GPU. 

Also inspired by DLSS Enabler, this should allow above x2 NATIVE MFG in (at least) Ada GPUs.

In game settings show up to x4, can be overriden to 6x using Nvidia Profile Inspector.

Tested with a RTX4070S
